Tecnologia utilizadas:
MongoDB
Express
Angular
Node

para rodar o projeto necessario instalação do MongoDB
no promt de comando executar: MongoDB

necessario o node instalado na maquina
para rodar o backend acessar a pasta local via prompt de comando e executar: 
npm run dev

para rodar o angular1 acessar a pasta local via prompt de comando e executar:
npm run dev

projeto utilizara a porta 3000 e abrira automaticamente no navegador padrão
apos os builds
